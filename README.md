# Desafio de Projeto DIO
## Curso Java

### Sistema de Banco

Ideia do projeto, sistema de banco.

Foram criados classes e metodos e, em conjunto, tem como objetivo de fazer as ações de mostrar extrato, sacar, depositar e transferir.
Foi usado o JOptionPane para uma iterativadade com o cliente.

Obs.: Não tive muito tempo para organizar o sistema e tratar as exceções de erro.
Tem mais coisa que podemos acrescentar. Mas o meu tempo não permite (Trabalho).
Caso quem queira, verificar e estudar os tratamentos de erro. Fica a vontade.
